# Reciclagames

Plataforma de jogos educativos com temática de sustentabilidade e reciclagem, desenvolvida como projeto da disciplina de Informática e Educação IV.

## Sobre o Projeto

O Reciclagames reúne jogos interativos voltados à conscientização ambiental, tornando o aprendizado sobre reciclagem e sustentabilidade mais engajante. O primeiro jogo disponível é o **Eco-Tycoon**, onde o jogador gerencia recursos energéticos e sustentáveis.

## Jogos

### Eco-Tycoon
Simulador de gestão em que o jogador constrói e administra uma cidade sustentável, tomando decisões sobre energia solar, reciclagem e uso consciente dos recursos naturais.

## Tecnologias

- HTML5
- CSS3 com [Tailwind CSS](https://tailwindcss.com/)
- JavaScript (Vanilla)

## Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/carlosiilva66/reciclagames_IE-IV.git
   ```
2. Abra o arquivo `index.html` diretamente no navegador ou utilize uma extensão como o **Live Server** no VS Code.
